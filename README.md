# Шайтан-Программа для обработки музыкальной библиотеки

На самом деле это эксперимент по работе с нейросетями. Собственно данный код и является продуктом их жизнедеятельности. Вайб-код, если хотите



В INI файл вынесены пара настроек:

* Путь оригинальной библиотеки
* Путь обработанной библиотеки
* Путь для дубликатов
* Путь к программе для создания цифровых отпечатков музыки
